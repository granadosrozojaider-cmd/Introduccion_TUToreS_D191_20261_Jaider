# 🎓 TutoHub - Gestión de Tutorías Académicas

**TutoHub** es una plataforma web desarrollada para facilitar la conexión entre estudiantes y tutores. Permite la gestión de solicitudes, el seguimiento de estados y la visualización de datos mediante un panel administrativo moderno.

## 🚀 Características Principales

- **Sistema de Autenticación:** Registro e inicio de sesión seguro para Estudiantes y Tutores.
- **Dashboard Administrativo:** Control total sobre los usuarios registrados y las tutorías del sistema.
- **Gráficas en Tiempo Real:** Visualización de la distribución de usuarios mediante gráficas de dona.
- **Interfaz Moderna:** Diseño "Dark Mode" optimizado con Bootstrap 5.
- **Alertas Personalizadas:** Confirmaciones de acciones sensibles mediante SweetAlert2.

## 🛠️ Tecnologías Utilizadas

- Frontend: HTML5, CSS3, JavaScript (ES6+).
- Framework UI: [Bootstrap 5](https://getbootstrap.com/).
- Backend as a Service: [Firebase](https://firebase.google.com/) (Firestore & Authentication).
- Gráficas: [Chart.js](https://www.chartjs.org/).
- Componentes de Alerta: [SweetAlert2](https://sweetalert2.github.io/).

## 📂 Estructura del Proyecto

- `/public`: Vistas principales y dashboards.
- `/js`: Lógica de conexión a base de datos y administración.
- `/css`: Recursos visuales y estilos personalizados.
